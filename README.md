# Institute Manager App

Mobile-first offline web app for small computer institutes.

## Run
1. Open `index.html` in any modern browser (Chrome/Edge/Firefox).
2. Login with:
   - Username: `admin`
   - Password: `admin123`

## Modules Included
- Dashboard with KPIs and charts
- Student management (add/edit/delete/search/filter)
- Fee collection + pending/overdue tracking + receipt print/PDF
- Attendance marking + monthly attendance report
- Batch management
- Course management
- Reports export (CSV/Excel-compatible) and print/PDF
- Reminder support (WhatsApp link + SMS copy)
- ID card generator (print/PDF)
- Certificate generator (print/PDF)
- Settings, dark mode, backup/restore

## Notes
- Data is stored in browser `localStorage`.
- For PDF, use browser print and choose "Save as PDF".
- For mobile use, host this folder on any static server and open from phone.
